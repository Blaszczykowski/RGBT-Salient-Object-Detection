The code is for paper "RGB-T Image Saliency Detection via Collaborative Graph Learning" 
by Zhengzheng Tu, Tian Xia, Chenglong Li, Xiaoxiao Wang, Yan Ma and Jin Tang
To appear in Proceedings of IEEE TRANSACTIONS ON MULTIMEDIA,2019.
written by Tian Xia
******************************************************************************************************************
The code is tested on Windows10 with MATLAB R2016a.
******************************************************************************************************************
Usage:
>put the test images(included RGB and Thermal image) into file 'test'
We extracted the FCN-32S feature of RGB and Thermal image( https://github.com/DUTFangXiang/ExtractFCNFeature)
>put the FCN-32S features into file 'FCN-feature'
>run 'demo.m'
